# 学生管理系统（C++）

本项目是一个基于 C++ 开发的学生管理系统，实现了对学生信息的增删改查等基本操作，适合学习 C++ 编程、面向对象思想以及文件操作等内容。

## 功能介绍

- 添加学生信息
- 删除学生信息
- 修改学生信息
- 查询学生信息
- 显示所有学生信息
- 数据持久化（文件读写）

## 环境要求

- C++11 或以上
- 支持的编译器（如 g++, Visual Studio, Clang 等）

## 使用方法

1. **编译代码**

   ```bash
   g++ main.cpp -o student_manager
   ```

2. **运行程序**

   ```bash
   ./student_manager
   ```

3. 根据菜单提示进行相关操作。
